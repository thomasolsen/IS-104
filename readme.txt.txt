hva kan javascript brukes til? 
Javascript er en funksjon som kan brukes til en rekke forskjellige hendelser i en html fil.
hva en javascript faktisk gjør er at den forsklarer nettleseren hva den skal gjøre. 
man kan for eksempel bytte farge på deler, eller hele html filen med en javascript kode. 
Javascript gjør funkjsoner til live. som for eksempel en dropdown meny. 
som når man trykker på en knapp så vil det skje noe. Med andre ord så trenger man javascript for og få en funksjonibel side.

elementer
http://www.tizag.com/cssT/cssid.php
http://css­tricks.com/the­difference­between­id­and­class/
Den store hovedforskjellen på class og ID er at, ID er noe man kun kan bruke en gang i en html fil, 
mens class er noe en kan bruke gjentatte ganger. 
Det vil si at man bør bruke class på noe som man vet man kommer til og bruke gjentatte ganger i filen, 
mens ID er noe man bør bruke til et engangstilfelle.

for en liten del av html dokumentet, som for eksempel inni en paragraf.
